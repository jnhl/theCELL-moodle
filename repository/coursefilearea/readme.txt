Coursefilearea Moodle 2.x repository plugin
-------------------------------------------

This Moodle repository plugin has been provided by Tim Williams of AutoTrain e-Learning (tmw@autotrain.org).
It is intended to try as closely as possible to replicate the functionality of the Moodle 1.x course
files area, including the original folder structure.

Users of this plugin are should also install the repository file management block which will provide proper
management and upload of files to the course files area.

This Moodle 2.x repository plugin should be installed into the 'repository' directory within your
Moodle installation.
